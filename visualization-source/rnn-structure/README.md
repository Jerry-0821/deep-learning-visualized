# RNN Structure

This visualization compares one-to-one, one-to-many, many-to-one, and many-to-many RNN layouts.

- Open in Colab: `rnn_structure_colab.py`
- Outputs HTML: Yes, the Python wrapper displays the visualization and writes `rnn-structure.html` inside the Colab runtime.
- Website connection: Source/reference code for the website topic `RNN Structure`; the deployed website uses the exported prototype in `public/prototypes/`.
