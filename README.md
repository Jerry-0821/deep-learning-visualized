# Deep Learning Visualized

A refined visual learning space for deep learning concepts, with interactive topic pages and supporting visual notes.

## Tech Stack

- Next.js
- React
- TypeScript

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production Build

```bash
npm run build
npm run start
```

## Deploying To Vercel

This project is ready for Vercel's automatic Next.js detection.

- Framework Preset: `Next.js`
- Install Command: `npm install`
- Build Command: `npm run build`
- Output Directory: leave empty / default

After this repository is pushed to GitHub, import it in Vercel from:

```text
Jerry-0821/deep-learning-visualized
```
