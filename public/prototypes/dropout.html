Inverted Dropout
(PyTorch model.train() Mode)

Random masks per iteration → Prevents co-adaptation → Inverted scaling by 1/(1-p)

Masking Random per iteration
Purpose Prevents co-adaptation
Scaling Multiply by 1/(1-p)
Context model.train()
Key concepts
What to watch for
Random masks per iteration Each training pass samples a new Bernoulli mask, so different hidden units are dropped each time.
Prevents co-adaptation The network cannot rely on one exact pathway staying available, so features become more robust.
Inverted scaling by 1/(1-p) Active units are scaled during training so the expected activation level stays consistent.
