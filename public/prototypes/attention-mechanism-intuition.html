What to watch for
Core beginner intuition
Tokens become vectors Words are not processed as raw text. Each token is turned into a compact numeric representation that the model can manipulate.
Attention is selective lookup The focus token compares its query with earlier keys, then turns those scores into normalized weights with softmax.
Prediction uses updated context The final position combines what it already knows with what attention retrieved, then pushes that richer state toward likely next tokens.
